# Exercise Library

This reference provides a list of exercises organized by muscle group and equipment type, including minimal equipment alternatives.

## Chest (Horizontal Push)
- **Barbell Bench Press** (Commercial Gym)
- **Dumbbell Bench Press** (Home/Dumbbells)
- **Push-ups** (Minimal Equipment)
- **Incline DB Press** (Upper Chest focus)

## Back (Vertical/Horizontal Pull)
- **Pull-ups / Lat Pulldowns** (Vertical Pull)
- **Barbell Rows / DB Rows** (Horizontal Pull)
- **Seated Cable Rows** (Commercial Gym)
- **Inverted Rows** (Minimal Equipment)

## Quads (Squat Pattern)
- **Barbell Back Squat / Front Squat** (Commercial Gym)
- **Goblet Squat** (Home/Dumbbells)
- **Bulgarian Split Squat** (Minimal Equipment/Dumbbells)
- **Leg Press** (Commercial Gym)

## Hamstrings/Glutes (Hip Hinge)
- **Conventional / Romanian Deadlift** (Barbell)
- **DB Romanian Deadlift** (Dumbbells)
- **Glute Bridge / Hip Thrust** (Minimal Equipment)
- **Leg Curls** (Commercial Gym)

## Shoulders (Vertical Push / Lateral Delts)
- **Overhead Press** (Barbell/Dumbbells)
- **Lateral Raises** (Dumbbells/Cables)
- **Face Pulls** (Rear Delts)

## Arms
- **Biceps:** DB Curls, Hammer Curls, Barbell Curls
- **Triceps:** Skullcrushers, Tricep Pushdowns, Dips

## Minimal Equipment Alternatives
- **Squats:** Bodyweight Squats, Lunges
- **Push:** Push-ups (various angles)
- **Pull:** Doorway Rows, Towel Pull-ups
- **Hinge:** Single-leg RDL (bodyweight)
